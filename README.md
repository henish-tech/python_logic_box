# python_logic_box
Python Logic Box Pattern Program
📌 Project Overview

This project demonstrates how to create box (square) patterns using Python. It is a basic logic-building program commonly used in practical exams and coding practice.

The program prints different types of box patterns using loops and conditions.

🎯 Objectives
Understand loop concepts in Python
Build logic using nested loops
Practice pattern-based programming
Improve problem-solving skills
🧠 Concepts Used
for loop
Nested loops
Conditional statements (if-else)
User input
